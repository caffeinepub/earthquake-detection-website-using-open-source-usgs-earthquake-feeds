# Deployment Runbook

This document describes how to deploy the WhoFeelAnEarthquake application to the Internet Computer.

## Prerequisites

Before deploying, ensure you have:
- **dfx** (Internet Computer SDK) installed and configured
- **Node.js** and **pnpm** installed
- **Internet Computer wallet** with sufficient cycles for deployment
- **Network connectivity** to the Internet Computer mainnet or testnet
